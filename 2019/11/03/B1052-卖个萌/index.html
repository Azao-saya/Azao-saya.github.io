B1052-卖个萌

2019年11月03日 650 字 大概 3 分钟 PAT练习

题目描述

萌萌哒表情符号通常由“手”、“眼”、“口”三个主要部分组成。简单起见，我们假设一个表情符号是按下列格式输出的：

[左手]([左眼][口][右眼])[右手]

现给出可选用的符号集合，请你按用户的要求输出表情。

输入描述

输入首先在前三行顺序对应给出手、眼、口的可选符号集。每个符号括在一对方括号 []内。题目保证每个集合都至少有一个符号，并不超过 10 个符号；每个符号包含 1 到 4 个非空字符。

之后一行给出一个正整数 K，为用户请求的个数。随后 K 行，每行给出一个用户的符号选择，顺序为左手、左眼、口、右眼、右手——这里只给出符号在相应集合中的序号（从 1 开始），数字间以空格分隔。

输出描述:

对每个用户请求，在一行中输出生成的表情。若用户选择的序号不存在，则输出 Are you kidding me? @\/@。

示例

输入

[╮][╭][o][~\][/~]  [<][>]
 [╯][╰][^][-][=][>][<][@][⊙]
[Д][▽][_][ε][^]  ...
4
1 1 2 2 2
6 8 1 5 5
3 3 4 3 3
2 10 3 9 3

输出

╮(╯▽╰)╭
<(@Д=)/~
o(^ε^)o
Are you kidding me? @/@

解题思路：

  • 只得了4分
  • 这题柳神的代码也是错误的网上也没别人的ac代码，想了一下可能是一些转义字符不能通过。
  • 试了一下之后发现也是可以的，莫得办法了。
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.ArrayList;
import java.util.List;

/**
 * [左手]([左眼][口][右眼])[右手]
 * 前三行顺序对应给出手、眼、口的可选符号集
 * a1 b1 c b2 a2
 */
public class PAT_B1052 {
    public static void main(String[] args) throws IOException { //测试点0,1错误未找到网上java解法
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        List<String> hand = parse(br.readLine());
        List<String> eye = parse(br.readLine());
        List<String> mouth = parse(br.readLine());
        int n = Integer.parseInt(br.readLine());

        String[] datas=new String[n];
        for (int i = 0; i <n ; i++) { //不这么做会直接显示内容
            datas[i] = br.readLine();
        }
        for (int i = 0; i <n ; i++) {
            String[] temp =datas[i].split(" ");
            int[] num = new int[5];
            for (int j = 0; j <5 ; j++) {
                num[j] = Integer.parseInt(temp[j]);
            }
            try {
                System.out.printf("%s(%s%s%s)%s",hand.get(num[0]),eye.get(num[1]),mouth.get(num[2]),eye.get(num[3]),hand.get(num[4]));
            }catch(Exception e) {
                System.out.print("Are you kidding me? @\\/@");
            }
            if(i!=n-1) {
                System.out.print("\n");
            }
        }
    }
    public static List<String> parse(String s) {
        boolean isStart = false;
        List<String> list = new ArrayList<>();
        String temp = "";
        list.add(temp); //从1开始
        for (int i = 0; i < s.length(); i++) {
            if (s.charAt(i) == '[') {
                isStart = true;
            } else if (s.charAt(i) == ']') {
                isStart = false;
                list.add(temp);
                temp = "";
            } else {
                if (isStart) {
                    temp += s.charAt(i);
                }
            }
        }
        return list;
    }
}
