B1038-统计同成绩学生

题目描述

  • 本题要求读入 N 名学生的成绩，将获得某一给定分数的学生人数输出。

输入描述

 输入在第 1 行给出不超过 105 的正整数 *N*，即学生总人数。随后一行给出 *N* 名学生的百分制整数成绩，中间以空格分隔。最后一行给出要查询的分数个数 *K*（不超过 *N*的正整数），随后是 *K* 个分数，中间以空格分隔。

输出描述:

在一行中按查询顺序给出得分等于指定分数的学生人数，中间以空格分隔，但行末不得有多余空格。

示例

输入

10
60 75 90 55 75 99 82 90 75 50
3 75 90 88

输出

3 2 0

解题思路：

  • 最后一个测试点超时
  • 写博客的时候稍微改进了一下代码，前面几组测试大概减了10ms，但果然还是逃不出超时
  • 少了一组存储的数组和少了一次遍历还会超时那真是不知道要怎么做了
  • 存入一组成绩和一组要检查的成绩
  • 嵌套循环外面一层是要检查的成绩，里面一层是所有成绩
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;

public class PAT_B1038 {
    public static void main(String[] args) throws IOException { //最后一组超时，代码还可以优化，但java逃不过超时。
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        int n = Integer.parseInt(br.readLine());
        String[] grade = br.readLine().split(" ");
        String[] check = br.readLine().split(" ");
        StringBuilder sb = new StringBuilder();
//        int[] list = new int[Integer.parseInt(check[0])];
        for (int j = 1; j <check.length ; j++) {
            int count = 0;
            for (int i = 0; i <grade.length ; i++) {
                if (check[j].equals(grade[i])){
//                    list[j-1]++;
                    count++;
                }
            }
            sb.append(count+" ");
        }

//        for (int i : list) {
//            sb.append(i+" ");
//        }
        System.out.println(sb.substring(0,sb.length()-1));
    }
}
