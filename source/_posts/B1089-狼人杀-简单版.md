## 题目描述 <!--more-->

以下文字摘自《灵机一动·好玩的数学》：“狼人杀”游戏分为狼人、好人两大阵营。在一局“狼人杀”游戏中，1 号玩家说：“2 号是狼人”，2 号玩家说：“3 号是好人”，3 号玩家说：“4 号是狼人”，4 号玩家说：“5 号是好人”，5 号玩家说：“4 号是好人”。已知这 5 名玩家中有 2 人扮演狼人角色，有 2 人说的不是实话，有狼人撒谎但并不是所有狼人都在撒谎。扮演狼人角色的是哪两号玩家？

本题是这个问题的升级版：已知 *N* 名玩家中有 2 人扮演狼人角色，有 2 人说的不是实话，有狼人撒谎但并不是所有狼人都在撒谎。要求你找出扮演狼人角色的是哪几号玩家？

## 输入描述

>   输入在第一行中给出一个正整数 *N*（5≤*N*≤100）。随后 *N* 行，第 *i* 行给出第 *i* 号玩家说的话（1≤*i*≤*N*），即一个玩家编号，用正号表示好人，负号表示狼人。  

## 输出描述

>   如果有解，在一行中按递增顺序输出 2 个狼人的编号，其间以空格分隔，行首尾不得有多余空格。如果解不唯一，则输出最小序列解 —— 即对于两个序列 *A*=*a*[1],...,*a*[*M*] 和 *B*=*b*[1],...,*b*[*M*]，若存在 0≤*k*<*M* 使得 *a*[*i*]=*b*[*i*] （*i*≤*k*），且 *a*[*k*+1]<*b*[*k*+1]，则称序列 *A* 小于序列 *B*。若无解则输出 `No Solution`。 

## 示例

### 输入1

> 5
> -2
> +3
> -4
> +5
> +4

### 输出1

> 1 4

### 输入2

> 6
> +6
> +3
> +1
> -5
> -2
> +4

### 输出2

> 1 5

### 输入3

> 5
> -2
> -3
> -4
> -5
> -1

### 输出3

> No Solution

import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.Arrays;

/**
 * 已知条件
 * 首先，只有两个狼人，其余全是好人
 * 其次，只有两个人说谎
 * 最后，一个狼人说谎，一个好人说谎
 *
 * 思路大概就是我们设立双重循环，依次假设两个人是狼人（人为满足条件一），看是否可以满足条件二和三。
 *
 * 设置一个say数组存放每个人说的信息，正负代表是否为狼，一个identity数组存放每个人的身份信息正负代表是否为狼，一个lie数组存放说谎的人。
 *
 * 对于如何判定是否说谎，说谎有两种情况：
 *
 * 一是真实身份是狼人，被说成是好人
 * 二是真实身份是好人，被说成是狼人
 * 其他情况均为真话，故当异号时为说谎，同号为真。可以设置判断条件是当
 * say[i]∗identity[Math.abs(say[i])]&lt;0 say[i]*identity[Math.abs(say[i])]&lt;0say[i]∗identity[Math.abs(say[i])]<0 时，为说谎。
 * 这时候加入lie数组，然后数组中有且仅有两个人，同时
 * identity[lie[0]]+identity[lie[1]]==0 identity[lie[0]]+identity[lie[1]]==0identity[lie[0]]+identity[lie[1]]==0,，判断条件满足。
 *
 * 还有需要注意的一点是，解不唯一时选最小的，看上去很复杂，但是从小到大开始遍历，第一个满足条件的就是最小序列。

public class PAT_B1089 {
    public static void main(String[] args) throws IOException { //只能做到理解的别人的代码，自己完全写不了
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        int n = Integer.parseInt(br.readLine());
        int[] say = new int[n + 1];
        boolean find = false;
        for (int i = 1; i <= n; ++i) {
            say[i] = Integer.parseInt(br.readLine());
        }
        br.close();
    flag:for (int i = 1; i <=n ; i++) {
            for (int j = i+1; j <=n ; j++) {
                int identity[]=new int[n+1];//真实身份数组
                Arrays.fill(identity, 1); //填充数组，意为初始化所有人为好人
                identity [i] = identity[j]=-1;		//设i和j为狼人
                int count = 0;
                int lie[]=new int [100];
                //判断说话是否为真，说是好人，确实是好人为真，
                //说是狼人，确实是狼人为真，其他为假，则同号为真，异号为假
                for(int k = 1 ; k <= n ; k++){
                    if (say[k] * identity[Math.abs(say[k])] <0){
                        lie[count]=k;
                        count++;
                    }
                }
                //若仅有两个人说谎，且一个是狼人一个人好人，则成立
                if (count == 2 && identity[lie[0]] + identity[lie[1]] == 0){
                    System.out.print(i+" "+j);
                    find = true;
                    break flag;
                }
            }
        }
        if (!find){
            System.out.println("No Solution");
        }
    }
}
